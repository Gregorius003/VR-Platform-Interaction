# VR Platform Interaction – Unity Project

## Description
This project is a virtual reality scene developed with Unity using the XR Interaction Toolkit.

The scene includes an interactive platform that moves along the Z axis when the player presses a button.  
The player can move using keyboard controls (WASD) and interact with the lever using keyboard input.

**Note:**  
VR headset interaction and controller input were not implemented because a VR headset and controllers were not available in the university environment where the project was developed. For this reason, interaction is simulated through keyboard controls.

## Technologies
- Unity
- C#
- XR Interaction Toolkit (version 2.4.3)

## Features
- Moving platform system
- Interactive lever controlled with keyboard
- Player movement with Rigidbody
- Trigger-based platform parenting system

## Controls
- **W / A / S / D** → Move the player  
- **Arrow Up / Arrow Down** → Rotate the lever  
- **B** → Activate platform movement  

## Project Structure
- **Assets** → Scripts, scenes and prefabs  
- **Packages** → Unity dependencies  
- **ProjectSettings** → Project configuration  

## Requirements
- Unity (recommended version used for the project)
- XR Interaction Toolkit 2.4.3

## How to Run
1. Clone the repository
2. Open the project with Unity
3. Open the main scene
4. Press **Play**
